# Quick Start Guide - Rocket Landing Simulator

## Installation

```bash
# Install dependencies
pip install torch pytorch matplotlib numpy pygame

# Navigate to project
cd rocket_landing
```

## Three Visualization Modes

### 1️⃣ VISUAL MODE (Recommended for demos)
**Interactive Pygame visualization with particle effects**

```bash
# Simple launch
python backend/visual_simulate.py

# Or with launcher
python backend/launch_visual.py

# Or with runner
python sim_runner.py --mode visual
```

**What You See:**
- Real-time 30 FPS animation
- Orange/red thrust flames with flickering
- Blue particle trail behind rocket
- Green HUD panel with stats
- Brown dust particles on landing
- Green success or red crash message
- 5-second result display

**Controls:**
- ESC - Quit anytime
- Window close - Quit

**Perfect For:**
- Live presentations
- Portfolio demonstrations  
- Video content
- Real-time feedback

### 2️⃣ PLOT MODE (Research & Analysis)
**Static matplotlib plots for papers and reports**

```bash
# Generate trajectory plots
python sim_runner.py --mode plot

# Custom model
python sim_runner.py --mode plot --model models/ppo_ep4500.pt
```

**What You Get:**
- `simulation_trajectory.png` - 2 plots:
  - Height vs Timestep (with ground reference)
  - Velocity vs Timestep (with thrust markers)
- Professional formatting
- Publication-ready quality
- Labeled axes, titles, grid
- Color-coded success/failure

**Perfect For:**
- Research papers
- Technical reports
- Publication-quality figures
- Static documentation

### 3️⃣ HEADLESS MODE (Benchmarking)
**Fast batch execution without rendering**

```bash
# Single episode stats
python sim_runner.py --mode headless

# Batch run 100 episodes
python sim_runner.py --mode headless --count 100

# Different model
python sim_runner.py --mode headless --count 50 --model models/ppo_ep1000.pt
```

**What You Get:**
- Per-episode results (success/crash)
- Success rate percentage
- Landing velocity statistics (avg, min, max)
- Reward averages
- Step count analysis
- No graphics overhead
- Fast execution (50-200 FPS)

**Perfect For:**
- Performance benchmarking
- Statistical analysis
- Batch testing multiple models
- Automated testing
- CI/CD pipelines

## Troubleshooting

### "Cannot connect to X server" (Linux)
```bash
# Use plot mode instead
python sim_runner.py --mode plot

# Or headless
python sim_runner.py --mode headless
```

### "Module not found" error
```bash
# Make sure you're in the main directory
cd rocket_landing

# Try with full path
python sim_runner.py --mode visual
```

### Model not found error
```bash
# Check model exists
ls models/ppo_ep2400.pt

# Use available model
python sim_runner.py --mode visual --model models/ppo_ep1000.pt
```

### Low FPS / Performance
```bash
# Use headless mode (no graphics)
python sim_runner.py --mode headless

# Or reduce particles (edit visual_simulate.py):
ParticleSystem(max_particles=200)  # from 500
```

## Python API Usage

### Direct import (advanced):
```python
# Visual mode
from backend.visual_simulate import run_visual_simulation
run_visual_simulation("models/ppo_ep2400.pt")

# Plot mode
from backend.simulate import run_simulation, plot_trajectory
trajectory, success, vy, steps = run_simulation("models/ppo_ep2400.pt")
plot_trajectory(trajectory, success)

# Headless mode
from backend.env import RocketEnv
from backend.agent import PPOAgent
env = RocketEnv()
agent = PPOAgent(obs_dim=5, action_dim=4, hidden=128)
agent.load("models/ppo_ep2400.pt")
```

## What Each Visualization Shows

### Visual Mode
Shows the agent making real-time decisions:
1. Reads height, velocity, fuel from sensors
2. Neural network predicts action
3. Applies thrust or idle
4. Physics updates position
5. Flame particles show thrust
6. Trail shows velocity
7. HUD shows stats
8. Landing produces dust

### Plot Mode  
Shows the complete trajectory:
1. Height over time (smooth descent)
2. Velocity over time (deceleration curve)
3. Points where thrust was applied
4. Final landing velocity
5. Total duration

### Headless Mode
Shows the success statistics:
1. Success/crash ratio
2. Landing velocities
3. Rewards earned
4. Steps taken
5. Aggregate statistics

## Tips & Tricks

### Record a video
```bash
# Use screen recording software while running:
python sim_runner.py --mode visual
# Then record your screen
```

### Batch test models
```bash
for model in models/ppo_ep*.pt; do
    echo "Testing $model"
    python sim_runner.py --mode headless --count 10 --model $model
done
```

### Custom output
```bash
# Redirect output
python sim_runner.py --mode headless --count 100 > results.txt

# Process with tools
python sim_runner.py --mode headless --count 100 | grep "Landing"
```
